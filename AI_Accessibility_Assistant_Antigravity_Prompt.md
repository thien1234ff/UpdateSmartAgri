# AI Accessibility Assistant Prompt for Antigravity

## Overview

This Markdown document is a starter template for your **Antigravity**
prompt.

## Project Information

-   Framework: Next.js 15 (App Router)
-   Language: TypeScript
-   Styling: Tailwind CSS
-   Database: Firebase Firestore
-   Authentication: Firebase Authentication
-   AI: Ollama (Llama3)
-   Deployment: Vercel

## Goal

Implement an AI-powered Accessibility Assistant that complies with
**WCAG 2.2 AA** without breaking any existing functionality.

## Core Features

### Mode 1 -- Read Entire Page

-   Read page title
-   Read headings
-   Read paragraphs
-   Read buttons
-   Read tables
-   Read forms
-   Play / Pause / Resume / Stop

### Mode 2 -- Focus Reading

-   Announce focused element
-   Read labels
-   Read validation messages

### Mode 3 -- AI Page Assistant

-   Answer only using semantic DOM
-   Never hallucinate

### Mode 4 -- Smart Guidance

-   Guide users step by step

## Code Safety

-   Never modify existing business logic.
-   Never rename existing APIs.
-   Never remove existing components.
-   Never change database schema.
-   Never break existing features.
-   Implement new features as isolated modules.
-   Prefer creating new files over editing existing ones.
-   Maintain backward compatibility.
